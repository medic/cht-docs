[Model Context Protocol (MCP)](https://modelcontextprotocol.io/) servers provide a standardized way for AI agents and tools to access external data sources and capabilities. The CHT community maintains MCP servers that give AI assistants direct access to updated CHT documentation, GitHub issues, code examples, community knowledge and forum posts.
